# Introduction

**LeetCode** is a platform that helps users improve their programming skills through interview questions commonly used at the world's leading technology companies.

The essence of **LeetCode** is to provide IT knowledge and this knowledge is often used at **Google, Amazon, Microsoft, Meta, Netflix**...

**LeetCode** is a good place for programmers to test their algorithm skills. It's a tool to support knowledge improvement, not really a tool to help you with "tricks" to pass the interview round.

This repository contains several solutions to **LeetCode** problems written in [Rust](https://www.rust-lang.org/), [Go](https://go.dev/), [Ruby](https://www.ruby-lang.org/en/), [Dart](https://dart.dev/), [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript), [C#](https://learn.microsoft.com/en-us/dotnet/csharp/). You can refer to it if you haven't found a solution yet.
